1. Installation Gemfiles
    - bundle install
2. Configure Database 
    Go to  config/database.yml and set the database credential.
3. Migrate Database
    - rake db:migrate
4. Start Rails Server
    - rails server
    Go to the browser and type localhost:3000 by default
